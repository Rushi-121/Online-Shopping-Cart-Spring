# Online-Shopping-Cart-Spring
The Online Shopping Cart is a console-based Java application developed using Spring Core (Annotation-based configuration) and JDBC. This project demonstrates CRUD operations (Create, Read, Update, Delete) on product data with proper layered architecture.

# Online Shopping Cart (Spring Core + JDBC)

## Technologies Used
- Java
- Spring Core (Annotation Based)
- JDBC
- MySQL
- Maven

## Features
- Add Product
- View All Products
- Update Product
- Delete Product

## Project Structure
- config → Spring Configuration
- model → Entity classes
- dao → Database operations
- service → Business logic
- user → MainApp (Console UI)

## How to Run
1. Import project in Eclipse
2. Configure database in DAO
3. Run `MainApp.java`
